# GameJam_26

## Tower of Boohbah (Tower of Babel)

## Requirements
- [ ] dungeon rouge crawler
- [ ] fight multiple enemies in waves per round
- [ ] Only have 3 hearts from the start
- [ ] defeat them by entering consecutive arrow buttons
  - [ ] different audios for each arrow and consecutively correclty inputing a combo
  - [ ] input consecutive list of arrows the press enter, finds which enemy has that combo of arrows and kill it. if not, nothing happens or a debuff occurs. if there are multiple enemies with same combo, kill all
- [ ] each round, gain a card boost that changes gameplay and enhace fighting/strategy capability
- [ ] cards that are activated by the user, will be added to the num bar which can activate by pressing a number
  - [ ] it includes reducing arrows to consectuively do, move enemies farther from player, shoot bullets to reduce arrows needed to select enemies, boost smth when inputing consecutive arrows correctly, set an ammount of enemies to have the same combo of arrows
  - [ ] cards to increase stats like increase max hearts, healing when defeatig enemies
  - [ ] cards that allow for consecutive arrows entered to effect all enemies who have that same order in their combo whether from the start, middle, or end of their combo
  - [ ] Card that removes the last arrow in a set amount of enemies
- [ ] Every 3 rounds, a boss battle, chiikawa, boohbahs and finally phill collins. rounds and a boss is considered a floor
  - [ ] boss works like inputting long consecutive arrows. highest enemy need 10 consecutive lines of arrows
  - [ ] some bosses will spawn smaller enemies instead of having one long line of arrows
